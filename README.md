# 🌴 Travel Destination Website - Phuket

A visually appealing **Travel Destination Website** showcasing the beauty, attractions, and experiences of Phuket. This project is built using **HTML, CSS, and JavaScript**, focusing on modern UI design and smooth user experience.

---

## 📌 Project Overview

This project is a static travel website designed to provide users with an immersive experience of exploring Phuket. It includes sections like attractions, gallery, about, and contact, making it a complete destination showcase.

The goal of this project is to demonstrate **frontend development skills**, including layout design, responsiveness, and user-friendly navigation.
